LA COMMUNICATION DANS LE WEB

Le Web est le terme communément employé pour parler du World Wide Web, ou www, traduit en français par la toile d'araignée mondiale, à chaque croisement il y a un ordinateur différent. Il fait référence au système hypertexte fonctionnant sur le réseau informatique mondial Internet et qui permet la consultation, via un navigateur, de pages de sites web.

COMMENT FONCTIONNE CETTE COMMUNICATION ?

Dans la définition du web on a parler d’ordinateur différent relier entre eux par la toile d’araignée géante. On distingue deux types d’ordinateur: il y a d’un cote le client et de l’autre cote le serveur web.

LE CLIENT

Est appelé client aussi bien l'ordinateur depuis lequel les demandes sont envoyées que le logiciel qui contient les instructions relatives à la formulation des demandes et la personne qui opère les demandes(les utilisateur du site web) à un serveur.

LE SERVEUR WEB

L'expression «serveur Web» désigne le logiciel utilisé sur le serveur pour exécuter les requêtes HTTP, le protocole de communication employé sur le world wide web. Il a pour mission de servir les client parce que c’est lui qui possède le site web et qui le distribue.

LES LANGAGES

Pour chaque site web on utilise des langages qui nous permettent de communiquer entre les clients et les serveurs.
côté client les langages informatiques utilisés sont HTML, CSS et JavaScript il sont indispensable pour afficher un site web c’est eux qui s’occupe de la représentation. Le rôle du navigateur web est de lire ses langages et les traduire dans un affichage connu et lisible par n’importe quel utilisateur.
Côté serveur ce sont des langages qui sont exécutés sur la machine du serveur, les client ne sont pas capable de les lire parmi ces langages on peut citer PHP, Java, Python etc. ils génèrent des pages web.L'avantage d'un langage serveur est qu'il permet d'adapter le site à l'utilisateur d'après ses besoins.

QU'EST CE QUI PERMET CETTE LIAISON ENTRE CLENT ET SERVEUR ?

Pour faciliter leur interactions des protocoles de communication ont été inventé. Dans le cadre de la communication web elle se fait grâce au protocole http pour faire transiter des pages web entre le client et le serveur, c’est ce protocole qui définit les règles de communication entre deux ordinateur.
Le client fait une requête avec toute les informations nécessaire et le serveur quant à lui il fait son calcul avec PHP et renvoi la page web HTML.
