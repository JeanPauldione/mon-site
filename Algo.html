C'EST QUOI UN ALGORITHME ?

Un algorithme est une suite d’instruction qui permet la résolution d’un problème à l’aide d’un ordinateur. Il s’agit de simplifier un problème complexe en ayant une suite d’opérations qui sont simplifiées pour arriver à un résultat. Il se découpe en trois phases :

  1. Les données (entrer): se sont les données fournis par l’énoncé d’un problème
  2. Le traitement (travail) : c’est-à-dire les opérations qu’on inflige aux au données récoltées, donc les suites d’instructions à mettre en place pour pouvoir résoudre un problème.
  3. Les résultats (sortie) : une fois que les opérations nécessaire effectuées c’est la résolution des problèmes.
En effet les algorithmes n’existent pas uniquement en informatique, ils existent depuis des années.
Exemple: Dans une recette de cuisine : on utilise un certain nombre d’ingrédient (début, variable initiale) et en suivant des étapes (processus, instruction) on obtient un résultat (variable finale) qui est le « thiébou dieune » ou un gâteau etc.
Pour résoudre un problème à l’aide d’un algorithme il y’a trois étapes à respecter.
  1. L’analyse du problème
  2. L’écriture de l’algorithme
  3. La traduction dans un langage de programmation
Donc on peut dire que l’algorithme est la base de tout programme car elle est indépendante de tout langage de programmation, elle permet de faire des opérations simples et élémentaires, et l’informatique en est son application.

LES VARIABLES

Une variable est un élément qui permet de stocker temporairement une valeur (texte, nombre, caractère etc.). Ces données ne sont pas figées, elles peuvent changer et varier lors de l’exécution du programme. Les variables sont la base de tout algorithme, car sans elles, il n’y à pas d’algorithme. Pour ce faire il faut savoir comment les nommées, les déclarées, les manipulées et les définir.

COMMENT NOMMER UNE VARIABLE

  • Une variable doit commencer par une lettre ou un underscore (_)
  • Elle ne doit pas contenir de caractères spéciaux ni d'espace
  • On peut utiliser des underscores(_) ou le signe $

LES TYPES DE VARIABLES

Nous avons:

  • Les entiers numériques (Int) exemple : âge = 14
  • Les nombre flottant, nombre à virgule (Flot) exemple : prix = 120,46
  • Les chaines de caractères (Str) exemple : âge = « 25 »
  • Les Booléens (Bool) exemple : true et false

QUELQUES TERMES ESSENTIELS A RETENIR

La boucle: Elle permet de répéter une action plusieurs fois. Pour stopper la boucle, il faut connaître le cran d’arrêt ou le nombre de répétitions.
L’itérateur ou compteur: C’est un nombre entier qui permet de compter le nombre de répétition d’une boucle. En général on l’appelle i.
L’instruction conditionnelle: Elle permet de vérifier certaines conditions avant d’exécuter une instruction.

LA BOUCLE POUR : FOR

C’est une boucle à utiliser lorsqu’on connaît le nombre d’itérations (le nombre de fois qu’on va répéter une boucle).

LA BOUCLE TANT QUE : WHILE

C’est une boucle à utiliser lorsqu’on connaît le cran d’arrêt de la boucle (c’est-à-dire lorsqu’on connaît une limite à ne pas dépasser).

LA BOUCLE SI ALORS SINON

C’est une boucle à utiliser lorsqu’on veut exécuter une instruction sous certaines conditions.
